# Security Policy

## Supported Versions

Only the latest stable release receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please report security issues or vulnerability findings by contacting the team directly at security@stadiumiq.ai. We will respond within 48 hours to acknowledge your report and discuss patch timelines.
